# 📊 Investments v2

Инвестиционный портал для отслеживания финансовых инструментов, новостей рынка и анализа ценных бумаг. Проект построен по архитектуре BFF (Backend for Frontend) для оптимальной работы с данными из различных источников.

---

## 🚀 Технологии

### Frontend
| Технология | Назначение |
|------------|------------|
| **React** | Библиотека для построения пользовательских интерфейсов |
| **TypeScript** | Типизация для надежности и масштабируемости кода |
| **RTK Query** | Управление состоянием и кэширование данных на клиенте |
| **Highcharts** | Визуализация данных и построение графиков |
| **Trading View** | Профессиональные графики для анализа ценных бумаг |

### Backend (BFF)
| Технология | Назначение |
|------------|------------|
| **Node.js** | Среда выполнения JavaScript |
| **Express** | Веб-фреймворк для построения API |
| **MongoDB** | База данных для хранения пользовательских данных |
| **Redis** | Кэширование для оптимизации производительности |

---

## 🏗 Архитектура и особенности

### BFF (Backend for Frontend)
Проект использует паттерн **BFF**, где бэкенд выступает прослойкой между фронтендом и внешними API. Это позволяет:

- ✅ Агрегировать данные из разных источников
- ✅ Оптимизировать данные специально для фронтенда
- ✅ Уменьшить количество запросов с клиента
- ✅ Скрыть детали реализации внешних API

### Кэширование
- **Новости на главной странице** — кэшируются для быстрой загрузки
- **Информация о ценных бумагах** — кэшируется как редко меняющиеся данные
- **Redis** — используется для высокопроизводительного кэширования

### База данных
**MongoDB** используется для реализации личного кабинета пользователя:

- Хранение профилей пользователей
- Сохранение избранных инструментов
- Сохранение настроек
- Гибкая схема данных для быстрого прототипирования

### Оптимизация данных
Бэкенд выполняет предварительную обработку и агрегацию данных перед отправкой на фронтенд, что позволяет:

- 📦 Уменьшить объем передаваемых данных
- 💻 Снизить нагрузку на клиентское устройство
- ⚡ Ускорить отрисовку интерфейсов

## ⚙️ Установка и запуск

### Предварительные требования
- Node.js (версия 20.18.2 или выше)
- MongoDB
- Redis

### Установка

```bash
# Клонирование репозитория
git clone https://github.com/yourusername/investments-bff.git

# Переход в директорию проекта
cd investments-bff

# Установка зависимостей для бэкенда
cd backend
npm install

# Установка зависимостей для фронтенда
cd frontend
npm install

```
### Настройка окружения
Создайте файл .env в папке backend:
```bash
BACKEND_PORT=your_port_number
BASE_URL=api_massive_url
ALT_BASE_URL=api_financial_modeling_prep_url
MONGO_URL=your_mongodb_connection_string
ALT_API_KEY=your_financial_modeling_prep_api_key
API_KEY=your_massive_api_key
JWT_SECRET="your_jwt_secret"
```
### Запуск
```bash
# Запуск бэкенда (из папки backend)
npm run dev

# Запуск фронтенда (из папки frontend)
npm run dev
```
